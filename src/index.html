Block Lambda

              
Block-Based λ-Calculus

            

          

          

            

              Block Lambda is a web-based IDE for experimenting with Lambda Calculus through
              Blockly-style visual blocks and generated textual code.
            

            

              Workspaces can be saved as .blc files and loaded again from disk.
              The IDE also keeps an automatic browser backup in local storage so recent work can
              be recovered if a manual save was forgotten.
